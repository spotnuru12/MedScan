# MedScan

A mobile app designed for ESL patients and older adults to understand their medications. Scan your prescription bottle, get simple explanations in your language. Built with **Expo**, **React Native**, and **Google Gemini**.

## Tech Stack

- **Google Gemini** (Vision + Text)

## Setup
```bash
git clone https://github.com/spotnuru12/MedScan.git
cd MedScan
npm install
cp .env.example .env
# Add your GEMINI_API_KEY to .env
npx expo start
```
